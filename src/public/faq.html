Editor Export Gallery FAQ
This app creates gradient palettes (atlases) for gradient texturing of 3D models. It's a perfect way to colorize low-poly assets without complex UV unwrapping.
Go to the Editor page and click on any gradient. Use the color pickers on the left to set the top and bottom colors for selected gradient.
Go to the Export page. You can right click the image to save it as a PNG or copy the link for further editing.
